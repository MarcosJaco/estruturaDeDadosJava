# Praticando Estrutura de dados em Java

## Objetivo:
Criar e reponder exercícios simples para fixar conceitos das seguintes estruturas de dados:

- **Array (vetor)**

- **Lista ligada (Linked List)**

- **Pilha (Stack)**

- **Fila (Queue)**

- **Árvore (Tree)**

- **Tabela hash (HashMap)**

- **Grafo (Graph)**
